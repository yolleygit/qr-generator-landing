# Product Overview

## QR Generator Landing Page (二维码生成器)

A minimalist, tech-focused single-page QR code generator that supports three modes:

- **Static QR**: Direct text/URL to QR code conversion
- **Dynamic QR (TOTP)**: Time-based one-time password QR codes (Google Authenticator compatible)
- **Encrypted QR**: AES-GCM encrypted QR codes with password protection

## Key Features

- **One-screen experience**: All functionality accessible without scrolling
- **Real-time generation**: Instant QR code updates as user types
- **Multiple formats**: PNG and SVG download options
- **Security-first**: WebCrypto API for encryption, TOTP RFC 6238 compliance
- **Responsive design**: Optimized for both desktop and mobile devices

## Target Users

- Developers needing quick QR code generation
- Security-conscious users requiring encrypted QR codes
- Anyone needing TOTP-compatible QR codes for 2FA setup
